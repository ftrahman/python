Method and Implementation 

Give a concise description of the implemented method. For example, you might describe the motivation of your idea, the algorithmic steps of your methods, or the mathematical formulation of your method. 

Briefly outline the functions you created in your code to carry out the algorithmic steps you described earlier. 

My functions greatly mirror those of the logistic regression. The differences include that the neural network should have more than two variables, there should be one including the hidden layer. Just as we initialize the input layer, we did the same for the hidden layer. 

In our forward propagation, we needed to find the dot product between the input layer and the hidden layer and apply the sigmoid activation function. We also did the same from the hidden layer to the output layer, this way we are tracing through the neural network in a forward direction. The soft max scores are then meant to constrict our outputs between 0 and 1 and enforce positivity. 

Once we have our outputs from forward propagation, we can then implement back-propagation to properly weigh our links. So we work backwards from the output to the input layer. Again, we need to apply a function to the outputs, so we use the derivative of the sigmoid. We do this from the outputs to the hidden layer, and again from the hidden layer to the input. 

Then we calculate the gradients of our functions similarly to the logistic regression, but we need twice the number of gradients (since we have an added input), and then finally update our gradients by applying an alpha function to increment the changes very gradually. 

Of course, we wrote a sigmoid and a sigmoid prime function to help aid in the activation functions.

The next major piece of code we implemented was for the cross validation. We take the data written in from the csv file and, knowing the length, divide the data into a list of 5 lists of equal size. Then, saving the first of the five data sets as our validation set, we only train the model for the other four data sets. We implemented a cumulative counter for the confusion matrices to get the average of all of the four data sets. We also average out the accuracies of each model.

Then we used our trained model on our validation set. 

The L2 regularization creates an additional bound to increase accuracy. For this, all we did was add the derivative of our regularization term to our gradient parameters.
